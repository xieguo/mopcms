环境信息

数据库大小： {$db_size}M
支持上传的最大文件： $post_max_size
是否允许打开远程连接： $allow_url_fopen
版本名称： $_M[sys][cmsname]
版本： $_M[sys][version]

webuploader一次最多传图上限： $_M[sys][webuploader_maxupload]张
JPG图片程序处理画质： $_M[sys][jpgquality]
图片加水印： {echo $_M[sys][watermark]? '是' : '否'}
远程图片本地化： {echo $_M[sys][rm_remote]? '是' : '否'}
文档是否允许评论： {echo $_M[sys][allow_comments]? '是' : '否'}
